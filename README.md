# 美食地圖 Food Radar 🍱

這是一個智慧美食搜尋工具，幫助你快速找到附近的餐廳和美食！

## 📁 專案檔案

### 1. `food-radar-bilingual.html` ⭐ **推薦使用**
**中泰雙語版本**
- 支援中文/泰文即時切換
- 右上角點擊「🇹🇭 ไทย」/「🇹🇼 中文」按鈕切換語言
- **適用場景：**
  - 在台灣找中式餐廳：切換中文搜尋
  - 在泰國找當地美食：切換泰文搜尋
  - 跨國旅行最方便！

### 2. `food-radar-chinese.html`
**純中文版本**
- 原始版本，全中文介面
- **適用場景：**
  - 在台灣或華語地區使用
  - 想找中式餐廳或華人經營的店家

### 3. `food-radar-thai.html`
**純泰文版本**
- 全泰文介面
- **適用場景：**
  - 在泰國找道地美食
  - 想找當地泰國人推薦的店家

## 🚀 使用方法

1. **選擇版本**：雙擊打開任一 HTML 檔案
2. **選擇模式**：「找正餐」或「下午茶」
3. **選擇情境**：隨便吃吃/1人/2人/多人
4. **選擇類型**：點擊你想吃的料理類型
5. **搜尋或隨機**：
   - 🔍 搜尋：直接搜尋你選的類型
   - 🎲 幫我選：讓命運決定！

## 🌍 工作原理

- 自動偵測你的**當前位置**（GPS定位）
- 在 **Google Maps** 搜尋附近的餐廳
- 根據語言顯示不同的搜尋結果：
  - 中文搜尋 → 找到中式/華人餐廳
  - 泰文搜尋 → 找到泰式/當地餐廳

## 💡 使用技巧

### 在台灣
- 使用**中文版**或**雙語版（切中文）**
- 搜尋「牛肉麵」、「火鍋」等，找到台灣本地店家

### 在泰國
- 使用**泰文版**或**雙語版（切泰文）**
- 搜尋「ก๋วยเตี๋ยวเนื้อ」、「อาหารไทย」等，找到道地泰國餐廳

### 跨國旅行
- 使用**雙語版**最方便
- 在不同國家切換不同語言搜尋

## 📊 功能特色

✅ 15種正餐類型（牛肉麵、火鍋、日式、韓式、泰式等）
✅ 8種下午茶類型（手搖飲、咖啡廳、炸雞、甜點等）
✅ 4種用餐情境（隨便吃、單人、約會、聚餐）
✅ 隨機選擇功能（選擇困難者救星！）
✅ 即時 Google Maps 導航
✅ 雙語切換支援
