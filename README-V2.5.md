# 美食地圖 Food Radar V2.5 🍱 ✨

## 🎉 全新優化版本！

這是升級版的美食雷達，新增了 5 大重要功能，讓使用體驗更棒！

---

## ⭐ V2.5 新功能

### 1. 🧠 **記住你的偏好**
- **自動記住語言選擇**：下次打開自動顯示你上次使用的語言（中文/泰文）
- **記住深色模式**：你的深色模式設定會被保存
- 使用瀏覽器 localStorage 技術，無需登入

### 2. 🌙 **深色模式**
- 點擊右上角的 🌙/☀️ 按鈕切換
- 保護眼睛，晚上使用更舒適
- OLED 螢幕更省電
- 所有顏色都經過精心調整，確保在深色模式下依然美觀

### 3. 🍱 **專業 Favicon**
- 瀏覽器分頁顯示可愛的便當圖示
- 更容易辨識和找到分頁
- 更專業的外觀

### 4. 📱 **分享優化（Open Graph）**
- 分享到 Facebook、LINE 時會顯示精美預覽
- 包含標題、描述和預覽圖
- 吸引更多朋友使用

### 5. 📲 **PWA 支援（像 App 一樣）**
- **可以安裝到手機主畫面**
- 像原生 App 一樣使用
- **離線也能開啟**（緩存技術）
- 不用下載，直接在瀏覽器安裝
- 支援 iOS 和 Android

## 🚀 使用方法

### 基本使用
1. 打開 `index-optimized.html`（推薦）
2. 選擇語言（中文/泰文）
3. 選擇模式（正餐/下午茶）
4. 選擇用餐情境
5. 選擇想吃的類型
6. 點擊搜尋或隨機選擇

### 安裝到手機（PWA）

**iOS (Safari):**
1. 打開網站
2. 點擊分享按鈕
3. 選擇「加入主畫面」
4. 完成！

**Android (Chrome):**
1. 打開網站
2. 點擊右上角 ⋮
3. 選擇「安裝應用程式」
4. 或者網站會自動彈出安裝提示
5. 完成！

## 💡 使用技巧

### 語言設定
- 第一次使用會是中文
- 切換到泰文後，下次打開會自動顯示泰文
- 適合經常在台灣和泰國之間移動的使用者

### 深色模式
- 晚上自動切換深色模式保護眼睛
- 設定會被記住，不用每次重新切換

### PWA 安裝
- 安裝後可以像 App 一樣使用
- 佔用空間極小（只有幾十 KB）
- 更新時會自動同步

## 🔧 技術細節

### 技術棧
- 純 HTML5 + CSS3 + Vanilla JavaScript
- 無框架依賴
- LocalStorage API（資料儲存）
- Service Worker API（離線支援）
- Web App Manifest（PWA）
- Open Graph Protocol（社群分享）

### 瀏覽器支援
- Chrome 67+
- Safari 11.1+
- Firefox 63+
- Edge 79+
- 所有現代手機瀏覽器

### 效能
- 首次載入：< 100KB
- 重複訪問：幾乎即時（緩存）
- 離線也能使用
